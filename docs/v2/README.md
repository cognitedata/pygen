# Pygen v2 Documentation

This directory contains documentation specific to Pygen v2.

## Structure

- **architecture/** - System architecture and design decisions
- **api-reference/** - API documentation (auto-generated)
- **guides/** - User guides and tutorials
- **migration/** - Migration guide from v1 to v2
- **examples/** - Code examples

As we build v2, documentation will be added here progressively:

### Phase 0 (Foundation)
- Project structure overview

### Phase 1 (Client Core)

- API models reference
- Authentication guide

### Phase 2 (Validation & IR)
- Validation rules documentation

### Phase 3+ (Generators & Beyond)
- Generator architecture
- Template customization
- Multi-language support
- Runtime API reference
